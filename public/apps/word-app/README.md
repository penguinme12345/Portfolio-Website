# Word App

A lightweight, local-first browser word processor.

## Features

- Rich text editing (bold, italic, underline)
- Text alignment controls
- New document and autosave to local storage
- Open `.txt`, `.md`, `.html` files
- Save current document as TXT or HTML

## Files

- `index.html` - app shell and controls
- `styles.css` - responsive UI styling
- `app.js` - editor behavior and file handling

## Run

Serve the folder with any static server and open `index.html`.
